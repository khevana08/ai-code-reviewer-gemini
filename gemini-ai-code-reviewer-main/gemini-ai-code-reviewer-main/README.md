# AI Code Reviewer using Gemini API

This project is an AI-powered code reviewer that automatically analyzes pull requests and provides intelligent feedback using Google Gemini API.

## Features
- Automated code review using AI
- Bug detection and suggestions
- Integration with GitHub Actions
- Improves code quality

## Tech Stack
- Python
- Gemini API
- GitHub Actions

## Future Improvements
- Add web-based UI for real-time code review
